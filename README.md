## Quick Start (2 Minutes)

### 1. Get a Free API Key

Visit: https://aistudio.google.com/app/apikey
- Click "Create API Key"
- Copy your key (it's free!)

### 2. Load the Extension

**Chrome/Edge/Brave:**
1. Open `chrome://extensions/` (or `edge://extensions/`)
2. Enable "Developer mode" (top right)
3. Click "Load unpacked"
4. Select the `GeminiText_Universal_v2` folder
5. Done! 🎉

**Firefox:**
1. Open `about:debugging#/runtime/this-firefox`
2. Click "Load Temporary Add-on"
3. Select any file from the extension folder
4. Done! 🎉

### 3. Add Your API Key

1. Click the GeminiText icon in your toolbar
2. Click the ⚙️ Settings button
3. Paste your API key
4. Click "Add"
5. Status should show "1 API key configured"

## How to Use

### Keyboard Method (Fastest)

Type any text, then add a command:
```
This is my text ?m
```

Press **Space** or **Enter** - text transforms instantly!

### Available Commands

| Command | Action |
|---------|--------|
| `?m` | Improve clarity, grammar, impact |
| `?r` | Rephrase with different wording |
| `?f` | Convert to formal tone |
| `?c` | Convert to casual tone |
| `?s` | Shorten by ~40% |
| `?l` | Expand with details |
| `?t` | Translate to your language |
| `?e` | Explain in simple terms |

### Right-Click Method

1. Select any text on a webpage
2. Right-click → GeminiText → Choose command
3. Result copied to clipboard

## Troubleshooting

**"No API key configured"**
→ Add an API key in Settings (get free from Google AI Studio)

**Command not triggering**
→ Make sure you pressed Space/Enter after the command

**"API Error"**
→ Check your API key is valid, or add multiple keys for rotation

**Extension not showing**
→ Reload the page (Ctrl+R) or restart your browser
